Last Updated: January 7, 2026

Simple Summary: Saarthi does not collect, store, or transmit any personal data. Your privacy is 100% protected.

1. Information We Don't Collect

Saarthi does NOT collect:

  • Personal information (name, email, phone, etc.)
  • Browsing history
  • Search queries
  • Financial data
  • Location data
  • Any form of analytics or tracking

2. What We Access

Saarthi only accesses:

  • Current webpage text: To detect mutual fund names and show tooltips
  • Chrome local storage: To cache fund data for faster performance (stored only on your device)

3. Data Storage

The only data stored is:

  • Fund data cache: Stored locally on your device using Chrome's storage API
  • Duration: Cached for 5 minutes, then automatically cleared
  • Purpose: To make the extension faster on repeated visits

4. External Data Sources

Saarthi fetches mutual fund data from:

  • GitHub Pages: https://blusummit.github.io/saarthi/data/
  • Data Source: AMFI (Association of Mutual Funds in India) and MFApi.in
  • No tracking: These requests do not contain any personal information

5. Third-Party Services

Saarthi does NOT use:

  • Google Analytics
  • Facebook Pixel
  • Any advertising networks
  • Any tracking services

6. Permissions Explained

activeTab: Allows reading the current webpage to detect fund names

storage: Allows caching fund data locally for performance

host_permissions: Allows fetching fund data from our GitHub Pages database

7. Your Rights

Since we don't collect any data, there's nothing to:

  • Request access to
  • Request deletion of
  • Export or download

You can uninstall the extension anytime to remove all local cached data.

8. Children's Privacy

Saarthi does not knowingly collect information from anyone. The extension is suitable for all ages.

9. Changes to This Policy

We may update this policy occasionally. Any changes will be posted on this page with an updated "Last Updated" date.

10. Contact

Questions about this privacy policy?

Email: support@blusummitventures.com

GitHub: https://github.com/blusummit/blusummit.github.io

Bottom Line: Saarthi is a privacy-first extension. We don't collect data because we don't need it. Your information stays on your device.
