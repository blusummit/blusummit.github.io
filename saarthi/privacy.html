Privacy Policy

Saarthi - Mutual Fund Navigator

Introduction

Saarthi ("we", "our", or "the extension") is a Chrome browser extension developed by BluSummit Ventures that provides mutual fund information when users hover over fund names on websites. We are committed to protecting your privacy and being transparent about our data practices.

Detailed Data Practices

1. Page Content Processing

Saarthi scans the text content of web pages you visit to identify mutual fund names. This processing:

  • Happens entirely on your device
  • Is never transmitted to any external server
  • Is not stored after you leave the page

2. Mutual Fund Data Retrieval

When you hover over a detected fund name, Saarthi fetches fund information from:

  • Source: GitHub Pages (https://blusummit.github.io/saarthi/)
  • Data fetched: Fund returns, benchmark data, category information
  • What we send: Only the fund name you hovered over
  • What we DON'T send: Your identity, browsing history, or any personal information

3. Local Storage

Saarthi uses Chrome's local storage API to:

  • Cache fund data temporarily (for faster subsequent lookups)
  • Store your preferences (if any)

This data remains only on your device, is never transmitted to us or any third party, and can be cleared by uninstalling the extension or clearing browser data.

4. No Analytics or Tracking

Saarthi does NOT:

  • Use Google Analytics or any tracking service
  • Collect usage statistics
  • Track which websites you visit
  • Monitor your browsing behavior
  • Create user profiles

Third-Party Services

GitHub Pages

We host our mutual fund database on GitHub Pages. When Saarthi fetches fund data:

  • GitHub may log standard server access information (IP address, timestamp)
  • This is governed by GitHub's Privacy Policy
  • We do not have access to these logs

Data Security

  • All data transmission uses HTTPS encryption
  • No sensitive data is ever collected or transmitted
  • Page content processing happens locally in your browser's sandboxed environment

Your Rights and Choices

You can:

  • Disable the extension at any time via Chrome's extension settings
  • Uninstall the extension to remove all locally stored data
  • Clear cached data through Chrome's "Clear browsing data" feature

Children's Privacy

Saarthi does not knowingly collect any information from children under 13 years of age. The extension processes only publicly available mutual fund data.

Chrome Web Store Compliance

This extension complies with the Chrome Web Store Developer Program Policies regarding user data handling.
